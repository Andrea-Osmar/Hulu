## Hulu Clone

The website is a clone of the popular streaming platform Hulu. Its is built with
Next.js and tailwind css for practicing purpose.

## View it live

## Tech

- React.js
- Next.js
- Tailwind CSS

## Packages Used

- `Hero Icons`
- `React Flip Move`
- `Tailwind Scrollbar Hide`

## API
